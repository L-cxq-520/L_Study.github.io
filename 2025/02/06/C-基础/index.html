数组

一维数组

基本概念

  • 数组是存储一组相同类型数据的集合
  • 数组分为 一维，多维，交错数组
  • 一般情况 一维数组就简称为数组
  • 声明一个变量可以理解为开辟了一个房间，而声明一个数组可以理解为连续分配了多个房间。

数组的声明

  • 变量类型[] 数组名;//只是声明了一个数组，但是并没有初始化
  • 变量类型 可以是我们学过的或者没学过的所有变量类型
  • 变量类型[] 数组名 = new 变量类型[数组的长度];
  • 变量类型[] 数组名 = new 变量类型[数组的长度]{内容1, 内容2, 内容3, …….};
  • 变量类型[] 数组名 = new 变量类型[] { 内容1, 内容2, 内容3, …….};
  • 变量类型[] 数组名 = {内容1, 内容2, 内容3, …….};

数组的使用

  • 声明一个数组
1. 数组的长度
2.获取数组中的元素
  • 数组中的下标和索引，它们是从0开始的
  • 通过索引下标去获取数组中某一个元素的值时
  • 一定注意！！！！！
  • 不能越界——不能越界——不能越界
3.修改数组中的元素
4.遍历数组
  • 通过循环 快速获取数组中的每一个元素
5.增加数组的元素
  • 数组初始化以后 是不能够 直接添加新的元素的
6.删除数组的元素
  • 数组初始化以后 是不能够 直接删除元素的
7.查找数组中的元素
  • 要查找数组里面的元素在哪个位置
  • 只有通过遍历才能确定数组中是否储存了一个目标元素

总结

  • 概念：同一变量类型的数据集合。
  • 一定要掌握的知识：声明，遍历，增删查改。
  • 所有的变量类型都可以声明为数组。
  • 数组是用来批量存储游戏中的同一类型对象的容器，比如所有的怪物、所有玩家。

二维数组

基本概念

  • 二维数组是使用两个下标(索引)来确定元素的数组。
  • 两个下标可以理解成行标和列标。
  • 可以用二维数组 int[2, 3] 表示，好比两行三列的数据集合。
  • 声明一个二维数组可以理解为连续分配了 行 x 列 个房间。

二维数组的声明

  • 变量类型[,] 二维数组变量名;
  • 变量类型[,] 二维数组变量名=new 变量行[行，列];
  • 变量类型[,] 二维数组变量名=new 变量行[行，列]{ {0行内容1, 0行内容2, 0行内容3…}, {1行内容1, 1行内容2, 1行内容3…}, {2行内容1, 2行内容2, 2行内容3…}…};
  • 变量类型[,] 二维数组变量名 = new 变量类型[,]{ {0行内容1, 0行内容2, 0行内容3…}, {1行内容1, 1行内容2, 1行内容3…}, {2行内容1, 2行内容2, 2行内容3…}…};
  • 变量类型[,] 二维数组变量名 = { {0行内容1, 0行内容2, 0行内容3…}, {1行内容1, 1行内容2, 1行内容3…}…};

二维数组的使用

  • 声明一个二维数组
1.二维数组的长度
  • 我们要获取 行和列分别是多长
2.获取二维数组的元素
  • 注意：第一个元素的索引是0 最后一个元素的索引肯定是长度-1
3.修改二维数组中的元素
4.遍历二维数组
5.增加数组的元素
  • 数组 声明初始化过后，就不能再原有的基础上进行添加或者删除了
6.删除数组的元素
  • 删除上面增加的元素
7.查找数组的元素
  • 如果要在数组中查找一个元素是否等于某个值
  • 通过便利的形式去查找

总结

  • 概念：同一变量类型的行列数据集合。
  • 一定要掌握的内容：声明，遍历，增删查改。
  • 所有的变量类型都可以声明为二维数组。
  • 游戏中一般用来存储矩阵，在控制台小游戏中可以用二维数组来表示地图格子。

在使用和储存上的区别

在使用上的区别

  • 值类型在相互赋值时 把内容拷贝给了对方 它变我不变
  • 引用类型的相互复制 是 让两者指向同一个值 它变我也变
为什么有以上区别
  • 值类型和引用类型储存在的内存区域是不同的，储存方式是不同的
  • 所有就造就了它们在使用上的区别

在储存上的区别

  • 值类型储存在栈空间——系统分配，自动回收，小而快
  • 引用类型储存在堆空间——手动申请和释放 大而慢
  • new了就是开了新的房间（地址）和之前的 没有什么关系了，所以arr不会有任何变化

练习题

1.请问打印结果为多少？

  • 解释: int 是值类型，将 a 的值（10）复制给 b 时，是将实际的值拷贝，而不是引用。因此，b = 20; 不会影响到 a 的值。

  • 答: 20
  • 解释: int[] 是引用类型，a 和 b 指向同一块内存地址。修改 b[0] 的值会影响到 a[0]，因为它们指向相同的数组。
3.请问打印结果为多少？

  • 解释: string 是一个特殊的引用类型，但它是不可变的。在这个例子中，str 和 str2 最初都指向 “123” 这个字符串，但在 str2 = “321”; 时，str 不受影响，因为字符串是不可变的，新的字符串 “321” 会被分配到一个新的内存地址，而 str2 将指向这个新的地址。

  • 可以使用函数（方法）的名称来执行该代码块

函数（方法）的主要作用

  • 2.提升代码复用率（少写点代码）

函数写（声明）在哪里

ref和out

学习ref和out的原因

  • 它们可以解决 在函数内部改变外部传入的内容 里面变了 外面也要变

ref和out的使用

  • 学习ref和out来解决上面的问题
  • ref 和 out 是函数参数的修饰符。
  • 当传入的值类型参数在内部修改时，或者引用类型参数在内部重新声明时，外部的值会发生变化。
ref
out

ref和out的区别

  • 1.ref传入的变量必须初始化 out不用
  • 2.out传入的变量必须在内部赋值 ref不用

总结

  • 1.ref和out的作用：解决值类型和引用类型在函数内部改值 或者重新声明 能够影响外部传入的变量 让其也被修改
  • 2.使用上：就是在声明参数的时候 前面加上ref和out的 关键字即可
  • 区别：
  • ref传入的变量必须初始化 但是在内部 可改可不改
  • out传入的变量不用初始化 但是在内部 必须修改该值（必须赋值）

{
    // 判断用户名是否正确

            info = "登录成功";
        }
        else
        {
            info = "密码错误";
            return false;
        }
    }
    else
    {
        info = "用户名错误";
        return false;
    }
    return true;
}
